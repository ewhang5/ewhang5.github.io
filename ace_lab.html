Handwritten Digit Identifier

This was my first deep learning model created at ACE Lab. I created a CNN that uses the MNIST handwriting datatset to identify handwritten numbers using Python and Keras. I used Conv2D, MaxPooling2D, Flatten, Dense, and Dropout layers with an Adam optimizer and sparse_categorical_crossentropy loss function for my model. My model achieved an accuracy of 98.38%. You can check out my code here.

Cats vs. Dogs Image Identifier

I created a CNN that used the cats_vs_dogs TensorFlow dataset to distinguish between images of cats and dogs. After importing the images, I shuffled and split the dataset to an 80:20 split between training and testing. I used the same layers, optimizer, and loss function from the MNIST CNN. You can check out my code here.

Predicting Character using Pen Trajectory

I created an LSTM that predicted the character being written using the trajectory of a pen. After importing the data, it was resized, reshaped, and shuffled. 67% of the data was used for training and the rest for testing. I created a Sequential model that used LSTM and Dense layers, an Adam optimizer, and categorical_crossentropy loss function. You can check out my code here.

Classifying Dry Bean

I created an autoencoder that classified the type of dry bean based on its characteristics like roundness or perimeter. I used Pandas DataFrame to import the data and split it into 80:20 training and testing datasets. I used Dense layers with a ReLU activation function for the encoder and a sigmoid activation function for the decoder. My model used an Adam optimizer and binary_crossentropy loss function. You can check out my code here.
